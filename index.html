Patient Context (MAS)
MAS Drug Screening
Drug interactions, disease contraindications, allergy warnings, duplicate therapy — full patient profile screening
Drugs are identified by code type (GCR, GFC, or NDC). Use the Drug Lookup tab to resolve a name to a code. NewDrugList = drugs being added. CurrentDrugList = drugs patient already takes.
Drug Lookup (LUSRequest)
Resolve a generic or brand name to a GCR code using the CKO LookUp service
Use GNGCRRT to look up by generic name keyword, or PRDGCRRT for brand/product names. Results give the GCR code needed for MAS screening.
